## Getting Started

Bienveue sur notre application de Gestion de projets MyProject.

## Folder Structure

Le dossier contient

- `data`: le dossier des ressources
- `src`: le dossier des class

# Compilation

Afin de compiler le projet il faudra lancer le jar fourni.
Dans le cas où le jar ne fonctionne pas je vous conseille de compiler tous les fichiers, puis d'executer le fichier Maiin voici les lignes de commandes à entrer :

javac src/Main/_.java src/Data/_.java src/InterfaceGraphique/_.java src/Model/_.java
java src/Main/Main
